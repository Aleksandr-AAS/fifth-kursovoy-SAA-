В этом разделе описывается, как проводить тестирование проекта.

### 3. Инструкции по запуску тестов

1.  Убедитесь, что у вас установлены все зависимости, указанные в разделе `Установка и настройка`.
2.  Запустите команду `pytest` в корневом каталоге проекта, чтобы выполнить все тесты.
3.  Чтобы запустить все тесты с отчетом о покрытии кода, выполните: `pytest --cov.

### 4. Результаты тестирования

*   Результаты тестов будут выведены в консоль.
*   Файлы отчетов о покрытии кода будут сгенерированы в каталоге `./htmlcov/`.
